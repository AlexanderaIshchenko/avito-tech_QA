# Тест-кейсы для тестирования счётчиков для десктопно версии.

## Тест-кейс 1
### Название: Отображение счетчиков при переходе на страницу неавторизованного пользователя

### Шаги
1) Зайти на страницу счетчиков на десктопной версии
2) Сделать скриншот текущего состояния счетчиков

### Ожидаемый результат: Счетчики серого цвета, все значения равны нулю

## Тест-кейс 2
### Название: Отображение счетчиков при значении 1

### Шаги
1) Зайти на страницу счётчиков на десктопной версии
2) Сделать подмену значений счётчиков на значения: 1
3) После каждой подмены сделать скриншот каждого счетчика

### Ожидаемый результат
1) Счетчики становятся черным
2) При переходе пограничных значений счетчики переводятся в более крупную систему измерения
